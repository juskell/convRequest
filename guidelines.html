Conversion Guidelines

SPEC Guidelines:

  • A SPEC conversion is processed through a SPEC Conversion Request when the changes desired require a config sheet to be filled out.
  • The exception to the above rule being shelving or SS leg swaps, which are done at dock level at time of shipment.
  • If the only change is from or to celsius or fahrenheit, the converison must be entered as a Conversion Request, not a SPEC Conversion Request.

Frequently Asked Questions:

  • Where can I inquire on the status of a conversion?
    • Please email NewConversionRequests@truemfg.com for any inquires.
  • What if the finished good I want does not have an item number set up?
    • Please contact PDM to have a new item number set up prior to entering the conversion.
